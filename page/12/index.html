国家大剧院初体验

发表于 2016-03-07 更新于 2016-03-08 分类于 生活 阅读次数：

去年写过一篇说说一代宗师，那是第二次去电影院，这次是我第二次到剧院。其实我想写的和上次差不多，这次看的是样式雷。

买票之前看过豆瓣，评价不高，权衡之下还是去看看。想起来，之前唯一看过的一部话剧还是初中时课堂上播的《暗恋桃花源》，那倒很是经典。

以下内容没有顺序安排

  • 先说样式雷，从大学以来开始看一些古建筑的书，多数也是消遣一类，但还是听说过样式雷家族。对于讲清朝尤其是清末的事情其实我并不太感兴趣，可是样式雷毕竟和古建筑有关。

  • 故事起于重修圆明园，家族的理想和国家的现实的冲突，大概是这部剧的主线。昆曲的剧本中我更喜欢桃花扇和长生殿，牡丹亭出来一两处外总觉得难读，这部剧的题材却是对我的口味。

  • 剧初的感觉似乎是要在内忧外患中停止，没想到最后是在进入了朝廷势力斗争中，雷氏真正的困难和觉醒竟来自两股势力的推波助澜，最后的解决也是势力的平衡和演变。但其中的争斗却又轻描淡写。

  • 感情戏似乎大概给剧中增添了色彩，可也没有展开得很好，整体上从相遇到成婚，几乎没有什么意外发生。吐衷肠的一幕大概是要表现小雷的木讷和纯粹，表演虽然逼真，雕琢却似乎不足，以至于有拖沓的感觉。

  • 戏剧中多有感情的演绎，牡丹亭的死可以生，桃花扇的血溅成画，长生殿的月宫相遇；而梁祝中的十八相送与这部剧的一幕极为类似，层层唱和虽然表达的意思重复，却一点不让人觉得多余。

  • 最终呈现出来的似乎有三条线：雷桂二人相恋，雷家梦想破灭，朝廷二党相争。就观感而言几乎同等重要，回头想来感情一线似乎又不过是个插曲。

  • 再说其他，舞台的布景极为精致，圆明园的断壁残垣和皇宫的金碧辉煌在顷刻间转变，人物于期间仿佛真的处在那个年代和场景中。

  • 剧中也有许多考究的地方，桂父挑担叫卖，桂枝儿舞者木马赶庙会，行人相见行礼，慈禧怀中的白猫，洋人鸣冤的轰动，无不透露着那个年代的习俗。

  • 剧中虽是一个沉重的话题，却多处透着幽默，初见太后时随口回答，雷家相会的试探，私定终身时的卿卿我我，照搬他人的说辞，乃至于官员上场，宫中奏对都不时让人会心一笑。

  • 总体说来这部剧很难让人想看第二次，没有太让人心动的地方，也似乎没有紧张的高潮，很多强烈的冲突都隐藏起来，藏得一点都不透露，画面精致但也定格不到某一幕，声音不时让人紧张起来却又很快消散。

不过，国家大剧院却有许多让人喜欢的地方。

读书偶记2016

发表于 2016-02-26 更新于 2016-02-28 分类于 读书笔记 阅读次数：

刚读完一本书——李碧研，《危机与重构：唐帝国及其地方诸侯》，北京师范大学出版社，2015年，这是我看到第一本涉及唐史的学术著作——但这不是重点，我真正想说的，是下面梳理出来的。

  • 早在中学时，我无意中读到一本顾颉刚的《秦汉的方士与儒生》，这本书对我后来选择书籍和读书的影响极大，与这个话题有关的是这样一个观点：上古的历史中，先出现三代的描述，而后是尧舜，再则是五帝，这段历史是后人层层加上的。

  • 大一大二的时候，看过一本小册子，是一个日本人写的和敦煌学有关学界往事之类的，里面提到他的老师内藤湖南，还有和前面说的顾先生书中类似的观点，似乎还提到唐宋变革。

  • 大二的时候选了一门课——域外汉学导论，需要选一本书写读后感，当时我选的是内藤湖南的《中国史学史》，实际上没有前面所说的，我可能还是会选这本书，因为其他的几乎都是文学理论一类的。

  • 在那本书里，内藤湖南提到了一点，大概是：唐及以前私家修史的传统，在唐宋开始没有延续下去。其中就包括了唐宋变革的一个方面，史学作为一种家学传统的改变。

  • 后来我查了一些唐宋变革的资料，但没有深入。大二末有一场宋史的讲座，那里又听到了内藤湖南和唐宋变革，知道了一些不一样的讨论。

  • 大概是一年多以前，偶然间在图书馆听了一场邓小南老师的讲座，讲的是宋朝，印象中是北宋，听完之后一直有一个想法，五代甚至唐代后期直接导致北宋的很多事实和制度。但是唐、宋的历史我其实都没怎么去了解过。

  • 回到这本书，其书以藩镇为视角，讨论的是唐在安史之乱后对中央与地方关系的重构，与此同时，地方的势力结构也发生变化。对我而言，从对于以上几条的延续：这本书描述了一些方面的“变革”，也展示了北宋所继承的某些部分。

我读书向来很随意，甚至可能是在学术上面没能发展的原因之一。即使在我的专业领域，也很少有这样长的时间延续着类似的关注点。上面的这些记录，包括读这本书，都有些有意，也有些无意。现在想来，我读书的关注点、包括对美术、建筑的兴趣点，大都在大四那一年开始变得比较集中，也许以后可以写写那一年。

北京出差记2015

发表于 2016-02-09 更新于 2016-12-18 分类于 生活 阅读次数：

我们部门去北京出差的机会是很少的，我的运气挺不错，11月份的时候就遇上一次机会。
工作上的安排是参加一场培训，最后留下了一个下午的时间，我也顺便完成了来北京之前的第二个计划。第一个计划已经完成，所以少了许多牵挂。
我最想去的是故宫，正值清明上河图展出，但是临时确定下来的一个下午显然去不了，所以改成了景山和北海。

上一次进景山公园只是穿过去，这次上了山顶，也仔细看了看崇祯皇帝的那棵树。

北海不曾进去过，但是那个白塔多少还是有些印象。到了秋天还有枯荷立在水中，园中假山以厚重的石头堆成，更显秋意。

毕业之前甚至很少会想起北京，工作这几个月来偶尔都会想起，这次去了北京一趟，以为会好了许多，却是更甚。更没有想到的是，11月底，单位决定让我到北京工作半年。

夏日闲逛在广州2015

发表于 2015-06-21 更新于 2016-12-11 分类于 游记 阅读次数：

在广州读书几年，很快就要离开到珠海去工作，这段时间正好也走不开，所以到处逛逛。

博物馆

广东省博物馆

没有去过南越王墓，粤博可以算得上是最好的一个，不过和上海、北京的博物馆比起来还是有些不足，展品看着有点少，几个常设展馆似乎有些单薄。（也很久没进去过，记不太清了）但是见过的临时展出却都不错，看过的明清画扇、宋元以来书画，展品涉及各个时代风格，布展也是合理，几乎每个展出都值得去一次。

南越王宫署博物馆

这也是广州最好的博物馆之一，展馆的一层是南越王宫署遗址，实际上是秦代南越王建国以来，各年代包括晋代、南朝、南汉等各个时期的官署、宫署遗迹。整个展馆最重要的展出就是这些遗迹，包括地下展馆和地面整理过的大殿遗迹，可以看出各个时代的建筑特点。遗址几乎涉及岭南历史各个重要的部分，了解岭南的历史不逊于粤博。

广州博物馆

广州博物馆在镇海楼，镇海楼原是广州城的北城，算是广州少见的重要古迹。市博物馆在馆藏上民俗的特色更多一点，但是展品没留下什么印象，大概也是因为布展方式的问题。我过去的时候正好有一个通草纸画的特展，通草纸作画多是画一些广州的民俗，笔法是西洋方式的，主要也是用于出口。虽然不少特别感兴趣，但这类展品也就是广州才会有。

广州艺术博物馆

广州艺术博物馆在白云山南麓，距离麓湖公园不远，有一些岭南地方特色的展品，比如佛山剪纸，马思聪展，这一点上倒和想想中的艺术博物馆还不太一样，庭院的设计，走廊的一些雕塑展品也是很有特点。这次去是为了看一个特展，展出山西出土的一批先秦玉器。上古时代的器物本就十分有趣，玉器用途广泛，先秦还有大量作为礼器使用，在广东看到这样的展出似乎是很难得的。


湖

麓湖

麓湖的麓指的是白云山南麓，那天正好从艺术博物馆出来，沿着湖边走，然后登白云山。湖光山色，景色自然还是不错，只是一直在广东生活，看惯了这种山山水水的样子，也不觉得有许多特别，湖面很大，最终我也只是从一侧走过。

海珠湖

我只在晚上去过海珠湖，离学校不远，是走这过去的，当然这是因为我想试试app的计步功能。就是一个接近郊区的公园，像广州这么热的地方，这种大片水面的公园，加上特地设计的步道，很是适合夏日的晚上散步。其实湖边还有一些活动的设施，建筑，还有一些可能是春天才有的大片的花，夜间看不清楚，但还是挺有趣的。

山

白云山

6年前我刚上大学的时候第一次来到广州，出去玩的第一个目的地就是白云山。当时大概还年轻，更是没有经历过多少旅行，早上上山，大中午就又饥又渴，还没爬山山顶就下来了。这次上山倒也没觉得有多远，想来也是，毕竟这也像是座大型的公园，向南望去可以看到广州城区。我从南门上山，西门下山，上山又热又累，下山时却遇上大雨，走在山间总有点担心一边的泥沙会不会掉下来，走到山下却又雨过天晴。

越秀公园

在广州好几年，也早知道越秀公园的五羊雕塑，但也仅有一次从越秀公园路过。这次特地上去逛一圈，广州博物馆也在上面。越秀山实际上是白云山的延伸，古代广州城的北城。山上有些古迹，除了镇海楼还有一段城墙、一座孙中山纪念塔，当时正好遇上下雨，还在塔下等了一段时间，这次走的似乎不在一个方向，并没有看见五羊雕塑。


晋国世系图

发表于 2015-05-17 更新于 2016-12-18 分类于 读书笔记 阅读次数：

这段时间在玩一个游戏（欧陆风云春秋mod），才注意到曲沃代翼，所以整理了一下晋国的历代国君。

晋国国君的传承在曲沃武公时第一次出现变动，来源于曲沃桓叔的壮大。第二次出现在晋献公时因为骊姬而改变继承人，直到晋文公即位才进入稳定的阶段。晋文公之后诸卿权力逐渐变大，向上可以追溯到曲沃代翼导致对于公室防范，晋献公时就处理了群公子，而晋文公时增加三行，增加了卿的数量。灵公、厉公均是诸卿所弑，悼公之后权力争夺转向诸卿之间，国君的权力进一步被蚕食，至出公被四卿所逐，知伯被三晋所杀，晋国已经实际上被三晋取代。

1 穆侯卒,弟殇叔自立,太子仇出奔。四年,穆侯太子仇率其徒袭殇叔，是为文侯。
2 曲沃武公伐晋侯缗,灭之。（周）釐王命曲沃武公为晋君,列为诸侯,于是尽并晋地而有之。
