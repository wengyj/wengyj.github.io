夏日闲逛在广州2015

发表于 2015-06-21 更新于 2016-12-11 分类于 游记 阅读次数：

在广州读书几年，很快就要离开到珠海去工作，这段时间正好也走不开，所以到处逛逛。

博物馆

广东省博物馆

没有去过南越王墓，粤博可以算得上是最好的一个，不过和上海、北京的博物馆比起来还是有些不足，展品看着有点少，几个常设展馆似乎有些单薄。（也很久没进去过，记不太清了）但是见过的临时展出却都不错，看过的明清画扇、宋元以来书画，展品涉及各个时代风格，布展也是合理，几乎每个展出都值得去一次。

南越王宫署博物馆

这也是广州最好的博物馆之一，展馆的一层是南越王宫署遗址，实际上是秦代南越王建国以来，各年代包括晋代、南朝、南汉等各个时期的官署、宫署遗迹。整个展馆最重要的展出就是这些遗迹，包括地下展馆和地面整理过的大殿遗迹，可以看出各个时代的建筑特点。遗址几乎涉及岭南历史各个重要的部分，了解岭南的历史不逊于粤博。

广州博物馆

广州博物馆在镇海楼，镇海楼原是广州城的北城，算是广州少见的重要古迹。市博物馆在馆藏上民俗的特色更多一点，但是展品没留下什么印象，大概也是因为布展方式的问题。我过去的时候正好有一个通草纸画的特展，通草纸作画多是画一些广州的民俗，笔法是西洋方式的，主要也是用于出口。虽然不少特别感兴趣，但这类展品也就是广州才会有。

广州艺术博物馆

广州艺术博物馆在白云山南麓，距离麓湖公园不远，有一些岭南地方特色的展品，比如佛山剪纸，马思聪展，这一点上倒和想想中的艺术博物馆还不太一样，庭院的设计，走廊的一些雕塑展品也是很有特点。这次去是为了看一个特展，展出山西出土的一批先秦玉器。上古时代的器物本就十分有趣，玉器用途广泛，先秦还有大量作为礼器使用，在广东看到这样的展出似乎是很难得的。


湖

麓湖

麓湖的麓指的是白云山南麓，那天正好从艺术博物馆出来，沿着湖边走，然后登白云山。湖光山色，景色自然还是不错，只是一直在广东生活，看惯了这种山山水水的样子，也不觉得有许多特别，湖面很大，最终我也只是从一侧走过。

海珠湖

我只在晚上去过海珠湖，离学校不远，是走这过去的，当然这是因为我想试试app的计步功能。就是一个接近郊区的公园，像广州这么热的地方，这种大片水面的公园，加上特地设计的步道，很是适合夏日的晚上散步。其实湖边还有一些活动的设施，建筑，还有一些可能是春天才有的大片的花，夜间看不清楚，但还是挺有趣的。

山

白云山

6年前我刚上大学的时候第一次来到广州，出去玩的第一个目的地就是白云山。当时大概还年轻，更是没有经历过多少旅行，早上上山，大中午就又饥又渴，还没爬山山顶就下来了。这次上山倒也没觉得有多远，想来也是，毕竟这也像是座大型的公园，向南望去可以看到广州城区。我从南门上山，西门下山，上山又热又累，下山时却遇上大雨，走在山间总有点担心一边的泥沙会不会掉下来，走到山下却又雨过天晴。

越秀公园

在广州好几年，也早知道越秀公园的五羊雕塑，但也仅有一次从越秀公园路过。这次特地上去逛一圈，广州博物馆也在上面。越秀山实际上是白云山的延伸，古代广州城的北城。山上有些古迹，除了镇海楼还有一段城墙、一座孙中山纪念塔，当时正好遇上下雨，还在塔下等了一段时间，这次走的似乎不在一个方向，并没有看见五羊雕塑。


晋国世系图

发表于 2015-05-17 更新于 2016-12-18 分类于 读书笔记 阅读次数：

这段时间在玩一个游戏（欧陆风云春秋mod），才注意到曲沃代翼，所以整理了一下晋国的历代国君。

晋国国君的传承在曲沃武公时第一次出现变动，来源于曲沃桓叔的壮大。第二次出现在晋献公时因为骊姬而改变继承人，直到晋文公即位才进入稳定的阶段。晋文公之后诸卿权力逐渐变大，向上可以追溯到曲沃代翼导致对于公室防范，晋献公时就处理了群公子，而晋文公时增加三行，增加了卿的数量。灵公、厉公均是诸卿所弑，悼公之后权力争夺转向诸卿之间，国君的权力进一步被蚕食，至出公被四卿所逐，知伯被三晋所杀，晋国已经实际上被三晋取代。

1 穆侯卒,弟殇叔自立,太子仇出奔。四年,穆侯太子仇率其徒袭殇叔，是为文侯。
2 曲沃武公伐晋侯缗,灭之。（周）釐王命曲沃武公为晋君,列为诸侯,于是尽并晋地而有之。

君主

发表于 2015-05-04 分类于 读书笔记 阅读次数：

本文根据《展望永恒帝国：战国时代的中国政治》（[以]尤锐，上海古籍出版社，2013）整理

《商君书·开塞》

天地设而民生之。当此之时也，民知其母而不知其父，其道亲亲而爱私。
亲亲则别，爱私则险。民众，而以别、险为务，则民乱。当此时也，民务胜而力征。务胜则争，力征则讼，讼而无正，则莫得其性也。故贤者立中正，设无私，而民说仁。当此时也，亲亲废，上贤立矣。
凡仁者以爱利为务，而贤者以相出为道。民众而无制，久而相出为道，则有乱。故圣人承之，作为土地、货财、男女之分。分定而无制，不可，故立禁；禁立而莫之司，不可，故立官；官设而莫之一，不可，故立君。既立君，则上贤废而贵贵立矣。

《吕氏春秋·恃君览》

凡人之性，爪牙不足以自守卫，肌肤不足以捍寒暑，筋骨不足以从利辟害，勇敢不足以却猛禁悍。然且犹裁万物，制禽兽，服狡虫，寒暑燥湿弗能害，不唯先有其备，而以群聚邪！群之可聚也，相与利之也。利之出於群也，君道立也。故君道立则利出於群，而人备可完矣。昔太古尝无君矣，其民聚生群处，知母不知父，无亲戚兄弟夫妻男女之别，无上下长幼之道，无进退揖让之礼，无衣服履带宫室畜积之便，无器械舟车城郭险阻之备。此无君之患。故君臣之义，不可不明也。自上世以来，天下亡国多矣，而君道不废者，天下之利也。故废其非君，而立其行君道者。君道何如？利而物利章。非滨之东，夷秽之乡，大解、陵鱼、其、鹿野、摇山、扬岛、大人之居，多无君；扬、汉之南，百越之际，敝凯诸、夫风、馀靡之地，缚娄、阳禺、驩兜之国，多无君；氐、羌、呼唐、离水之西，僰人、野人、篇笮之川，舟人、送龙、突人之乡，多无君；雁门之北，鹰隼、所鸷、须窥之国，饕餮、穷奇之地，叔逆之所，儋耳之居，多无君。此四方之无君者也。其民麋鹿禽兽，少者使长，长者畏壮，有力者贤，暴傲者尊，日夜相残，无时休息，以尽其类。圣人深见此患也，故为天下长虑，莫如置天子也；为一国长虑，莫如置君也。

《管子·内业》

天主正，地主平，人主安静。春秋冬夏，天之时也；山陵川谷，地之枝也；喜怒取予，人之谋也。是故圣人与时变而不化，从物而不移。能正能静，然后能定。定心在中，耳目聪明，四肢坚固，可以为精舍。精也者，气之精者也。气，道乃生，生乃思，思乃知，知乃止矣。凡心之形，过知失生。
一物能化谓之神，一事能变谓之智。化不易气，变不易智，唯执一之君子能为此乎！执一不失，能君万物。君子使物，不为物使，得一之理。治心在于中，治言出于口，治事加于人，然则天下治矣。一言得而天下服，一言定而天下听，公之谓也。

《吕氏春秋·孟春纪》

始生之者，天也；养成之者，人也。能养天之所生而勿撄之谓天子。天子之动也，以全天为故者也。

珠海深圳201504

发表于 2015-04-19 更新于 2016-12-18 分类于 游记 阅读次数：

论文交了稿，回家之前的一次短旅行。明天又要改稿了。

回到珠海

很有可能几个月后就去珠海工作了，本科那两年逛了不少地方，这次来到珠海，还是去那几个熟悉的地方。

在拱北下了轻轨，便想到了竹仙洞，还有最出名的桃花，但是这个时候已经过了花期。带着一丝丝的希望，还是到了竹仙洞，这次是直接上了桃园，果然只有两三朵花，我觉得像是桃花。好在一路上还是不缺盛开的花，比如开的红红火火的角花，还有一种好像是艳山姜。还有一种应该是兰花，查了一下似乎是巴西鸢尾兰，开着山间草丛里，颜色极为漂亮又带着几分平静。

第二天去的是景山公园，本来想看看港珠澳大桥，这里只能看到一段，倒是第二天在九洲港附近意外地看到了两个分岔。

景山似乎之前只上去过一次，记得当时觉得特别轻松，没想到这次还是有点累。

再到深圳

这是我第二次到深圳，上次没空到蛇口，这次直接从珠海过来，也是感受一下另外一条回家的路。

上次去荷包岛，在高栏港看到装满煤的大船，这次在蛇口看到的是成片的集装箱，本来想从赤湾的某座山上看看港口，没有成行，倒是见到了那艘法国的船。

原来预定竹子林的酒店不过是在蛇口和深圳北中间，没想到旁边居然有个园博园，作为园林算不算惊艳，当作一般散步的公园却有趣得多，3个小时下来还是没走完。

离开之前特地去了趟莲花山，本来想看看香港，不过山却是意外的矮，看来只能下次去看看太平山顶了。

戏曲选录

发表于 2015-01-23 更新于 2015-02-15 分类于 生活 阅读次数：

一直最喜欢桃花扇里的这一段：（余韵）

【北新水令】山松野草带花挑，猛抬头秣陵重到。残军留废垒，瘦马卧空壕；村郭萧条，城对着夕阳道。
【驻马听】野火频烧，护墓长楸多半焦。山羊群跑，守陵阿监几时逃。鸽翎蝠粪满堂抛，枯枝败叶当阶罩；谁祭扫，牧儿打碎龙碑帽。
【沈醉东风】横白玉八根柱倒，堕红泥半堵墙高。碎琉璃瓦片多，烂翡翠窗棂少。舞丹墀燕雀常朝，直入宫门一路蒿，住几个乞儿饿殍。
【折桂令】问秦淮旧日窗寮，破纸迎风，坏槛当潮，目断魂消。当年粉黛，何处笙箫? 罢灯船端阳不闹，收酒旗重九无聊。白鸟飘飘，绿水滔滔，嫩黄花有些蝶飞，新红叶无个人瞧。
【沽美酒】你记得跨青溪半里桥，旧红板没一条。秋水长天人过少，冷清清的落照，剩一树柳弯腰。
【太平令】行到那旧院门，何用轻敲，也不怕小犬哰哰。无非是枯井颓巢，不过些砖苔砌草。手种的花条柳梢，尽意儿采樵；这黑灰是谁家厨灶?
【离亭宴带歇拍煞】俺曾见金陵玉殿莺啼晓，秦淮水榭花开早，谁知道容易冰消!眼看他起朱楼，眼看他宴宾客，眼看他楼塌了！这青苔碧瓦堆，俺曾睡风流觉，将五十年兴亡看饱。那乌衣巷不姓王，莫愁湖鬼夜哭，凤凰台栖枭鸟。残山梦最真，旧境丢难掉，不信这舆图换稿！诌一套《哀江南》，放悲声唱到老。

这是千钟禄里的一段，有八支曲，但是我听起来还没太多感觉，比较喜欢这一支：（惨睹）

【倾杯玉芙蓉】收拾起大地山河一担装，四大皆空相。历尽了渺渺征途、漠漠平林、垒垒高山、滚滚长江，但见那寒云惨雾和愁织，受不尽苦雨凄风带怨长。雄城壮，看江山无恙， 谁识我一瓢一笠到襄阳。

玉簪记·琴挑这出戏很是有名气，情节挺有趣，前两天看到一个视频玉簪记·懒画眉，也是很有特点：

【懒画眉】月明云淡露华浓，倚枕愁听四壁蛩。伤秋宋玉赋西风。落叶惊残梦，闲步芳尘数落红。
【前腔】粉墙花影自重重，帘卷残荷水殿风，抱琴弹向月明中。香袅金猊动，人在蓬莱第几宫。 
【前腔】步虚声度许飞琼，乍听还疑别院风。凄凄楚楚那声中。谁家夜月琴三弄，细数离情曲未终。
【前腔】朱弦声杳恨溶溶，长叹空随几阵风。仙郎何处入帘栊，早是人惊恐。莫不是为听云水声寒一曲中。

说说一代宗师

发表于 2015-01-09 更新于 2015-01-18 分类于 生活 阅读次数：

我很少到电影院去看电影，准确地说这是第二次，也是第一次一个人去。大概几天前看到一代宗师的3d版要公映，顿时有点兴奋，因为原来的版本我在网上看了很多次，对于我这样很少看电影的人来说很是特别。当然，也是上次去电影院给我的体验还行，嗯，还行，应该是看比较典型的商业影片的那种感觉。

念念不忘，必有回响 我总觉得这句话很久之前就听过。

原版中另一句印象深刻的台词是， 相识半辈子，实则你不知她，她不知你

  • 大南的茶桌旁，宫二的浓妆下掩饰不了的病态，是另一种美，戏台下的低语，平静地诉说着似乎是她的一生。原版中我觉得是电影的高潮，3d版没了这样的感觉，但依然有不尽的回味。

      人生若无悔，那该多无趣
