2016年相片集

发表于 2017-02-04 更新于 2017-05-30 分类于 相册 阅读次数： 阅读次数：

2016年的主题是北京，期间也去了青岛和天津，不过青岛的两天天气不好，没有什么好照片。2015年底在景山和北海公园的两张也收了进来。


一直觉得秋天是北京最好的季节，恰好为只有秋天在北京的时间最少。这是12月初的照片，在泗上村附近的一个湖，当时已经是初冬了。


第二年遇见北京的雪，这是雪后的中午，和上一张是同一个地方，早了10天，还有些深秋的气息。


和上一张是同一个时间，满池的枫叶被吹到一角。


出差经常到的地方是莲花池附近，那是8月底特地过去一次，荷花的照片都不小心删掉了，这个角度的日落还不错。


这是5月初在凤凰山顶上，当时是单位的活动。


4月份病了一场，正值玉渊潭的樱花节，樱花很是漂亮，但没有拍好。好在见到难得的夕阳。


1月份刚到北京的时候，还不是很熟悉，第一次走得稍远是去八大处，也是冬天里少见的好天气。


到北京人艺看过那几次话剧，更多是在小剧场里，首都剧场实际上只去过一次，那是4月的时候，白天逛完故宫。


7月最后一次去北京人艺特地提前去看戏剧博物馆，这是茶馆的布景。


除了北京人艺，国家大剧院也去过几次，3月初第一次去的时候拍下的，一场歌剧的布景。


4月份去故宫，金水河果然使大广场多了几分灵动。


还是那次在故宫，自己逛的感觉好了很多，只是游人永远是那么多。


4月正好是花开的季节，其实更喜欢红墙和海棠相应，但是这一张是拍得比较好的。


故宫珍宝馆里，原是乾隆准备退位后住的地方，修得更近园林。


故宫里内部拍下的照片不多，这是一张内部装饰的照片。


钟表馆中的布景展示，钟表馆中钟表很多，这一间是展示当年钟表装饰场景的，似乎无处不少钟。


故宫建筑给人的影响莫过于三大殿的恢弘，但是后宫又更多相对轻松的装饰，这面墙正中用荷花做装饰。


有故宫不可错过景山公园，我多次上去过，最特别的一次是11月出差时遇见了下雪，屋檐上还留着雪。


景山公园是一处高点，许多人在那里看日落，这次没看到夕阳，但晚霞颇有意思。


秋天的雪后去的北海，还下着小雨，银杏叶子撒满山坡，满山的金黄。


颐和园，万寿山的北侧是四大部洲，都是藏传佛教的风格。


进入夏天才到颐和园去一次，天色很好，苏堤上漫步，时常可见飞鸟。


北宫门一进去就是苏州街，不过近几年新建的，风格上有些南北杂糅。


颐和园有一处名气很大的园中园——谐趣园，据说是仿造无锡寄畅园。


谐趣园中的濠濮间，典出庄子，也是常用的分割水面的手法，这个角度望去，正是一片翠绿。


香山原也是三山五园中的静宜园，园林依山而建，但也有几次不错的水景。


初春三月桃花绽开，圆明园大体上还是一片废墟，但桃红柳绿初现，多了几分生气。


五月再去圆明园，含元殿前的牡丹都已经凋谢了，正好有一朵还挂在枝头。


圆明园的舍卫城，本是一处城堡式的寺庙，如今正好是一片断壁残垣。


初春的柳树稍稍发芽，与断石、残荷相映，正是圆明园的趣味所在。


园博园中的北京园，远观有点颐和园佛香殿的感觉，园博园中除了园林博物馆，各地区的园做得都很不错，北京园算得上最好。


园林博物馆仿造了三个南方园林，这是扬州何园的假山，做得十分相似，扬州的假山看上去更有平远山水延绵起伏的感觉，这个则更显得高大。


仿苏州的畅园，一面长廊曲折，另一面也是长廊，但以墙隔开，开着几处漏窗。


同上，冰裂纹的圆形漏窗极为可爱。


园林博物馆另有三处北方风格园林，染霞山房依山而建，长廊没有南方园林那样灵动，漏窗也是另一种风格。


历代帝王庙的大殿，规格很高，游人很少。


太庙后殿的门墙，宫墙典型的黄瓦红墙，在阳光下十分显眼。


天坛皇穹宇外墙，以回音壁闻名，北京古建中最为特别的就是这些祭祀建筑。


十三陵神道的大门，叫大红门，十三陵三面靠山，有如一处天然风水，大红门也是陵区的大门。


十三陵神道以石刻著称，动物、人物各有特点。


长陵大殿现为展馆，这是定陵出土的金丝皇冠。


智化寺是东城的一座寺庙，因为保留了明代的宫廷音乐而显得尤为特别，这处壁画也是寺中一宝。


明城墙在崇文门一带，修整后的断壁残垣、树枯草黄，很有历史味道。


卢沟桥和石狮子因为卢沟桥事变变得更加出名，桥上狮子有些风化了许多，有些像是新的。


国家博物馆各类馆藏都能形成体系，以青铜器尤为出色。


国家博物馆的临时展更新频率大概适合每月过去一次，这是工艺美术双年展的展品。


国家博物馆新开的蜡像馆，虽然不大，但做得很有意思，这是梁思成和林徽因。


首博的海昏侯大展，最耀眼的自然是闪闪发光的黄金，这是罕见的马蹄金。


海昏侯大展同时还要妇好展，其实这个展更好一些，这个萌萌的是鸮形的酒壶。


北京动物园旁边有个真觉寺，又叫五塔寺，现在是石刻博物馆，现存建筑就是这个金刚座塔了。


妇人倚门石刻，似乎曾在书里见过。


石刻博物馆的石碑，北京的天气好起来，比任何地方都不逊色。


古代建筑博物馆藏的藻井，做出天宫的形象。


宿舍附近的宣南博物馆，富有印象中的北京民间气息，这是天桥八怪之一。


独乐寺山门，辽代建筑。


独乐寺观音阁，不管是中国建筑史还是介绍梁思成的书都会见到的一座建筑。


观音阁的牌匾，传闻是李白所写，署名“太白”。


辽代的庑殿顶，斗拱厚实，曲线优美，雄健而不失灵动。


蓟县除了古迹就是盘山，这是山中的飞泉。


晚霞映照下的蓟县，蓟县是座保留了北方特色的小县城，距离北京、天津城区都不太近。
